# Gradle Dependency

1. Add JitPack.io to your repositories list in the root projects build.gradle:

```gradle
repositories {
    maven { url "https://jitpack.io" }
}
```

2. Add the following to your module dependencies:

```gradle
dependencies {
    compile 'me.oriley:vista:0.2'
}
```
